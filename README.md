# Counter

A small personal side project: a simple counter built with React and TypeScript. Increment, decrement, or reset the value.

## Run it

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
